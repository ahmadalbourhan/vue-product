# Junior Developer Technical Task: "Product Catalog Lite"

## A. Setup Instructions

1. Initialize the project using `npm create vue@latest`
2. Clean up the boilerplate code.
3. Fetch product data from the API and render the list dynamically using `v-for`
4. Wrap `fetchProducts` in the `onMounted` hook to follow component lifecycle best practices.
5. Implement search functionality using `computed` properties for reactive filtering.
6. Fetch `categories` and implement reactive filtering using a dropdown menu
7. Handle view states using conditional rendering to toggle between product listing and detailed views.
8. Push to GitHub and deploy the application to Vercel for live hosting and continuous integration.
9. Install `tailwindcss` and `@tailwindcss/vite` via npm
10. Add the `@tailwindcss/vite` plugin to your Vite configuration.
11. Create `main.css` and add an `@import "tailwindcss";` to your CSS file that imports Tailwind CSS.
